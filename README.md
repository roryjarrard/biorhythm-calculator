# Biorhythm Calculator

A simple biorhythm calculator built with React and Ionic that calculates and visualizes physical, emotional, and intellectual cycles based on a user's birth date.

**Live Demo:** https://rnj-biorhythm-calculator.netlify.app/

---

## Overview

This project was built as part of learning the Ionic framework and React. It began as a guided tutorial but was completed into a fully functional application and deployed publicly. It provided hands-on experience with building mobile-friendly interfaces using modern frontend technologies.

---

## Features

- Calculate biorhythm cycles from a birth date
- Interactive date selection
- Visual chart of physical, emotional, and intellectual cycles
- Responsive layout
- Mobile-friendly interface

---

## Tech Stack

- React
- Ionic Framework
- TypeScript (if applicable)
- HTML/CSS
- Netlify (deployment)

---

## Running Locally

Clone the repository:

```bash
git clone https://github.com/roryjarrard/biorhythm-calculator.git
cd biorhythm-calculator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
